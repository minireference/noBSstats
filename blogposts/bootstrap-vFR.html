La statistique classique#

Les techniques de la statistique classique ont t dveloppes dans le temps o les capacits de calcul taient limites (le monde avait pas encore accs des ordis dans leur poches, dans leur sacs, et dans le cloud). Les statisticiens ont donc dvelopp des techniques simplifies au max. En commencent par une hypothses de modlisation pour la loi de probabilit pour la population inconnue, ils on fait de la modlisation mathmatique intense pour produire des procdures suivre : des recettes pour faire des analyses statistiques qui peuvent tre applique dans plein de cas. Ces recettes statistiques requirent lutilisation de certaines formulas mathmatiques, suivi de calculs de probabilit simple, qui peuvent tre effectus laide dun tableau de loi de probabilit imprims. Par exemple, la plupart des livres de stats incluent des tables ou sont imprims les valeurs des fonction de rpartition de quatre lois de probabilit centres et rduites: la loi normale \(Z\), loi \(t\) de Student \(T_\nu\), la loi du \(\chi^2\) (khi carr), et la loi \(F\) de Fisher.

En gros, les statisticiens du temps se disaient quil est mieux de rduire tout les calculs de probabilits au minimum. Ainsi, la statistique classique offre un API restreint incluant des procdures pour certains cas o ils peuvent dvelopper des formules se basant sur les quatre distributions de rfrence standard \(Z\), \(T_\nu\), \(\chi^2\) et \(F\). Au cours des cent dernires annes, les statisticiens ont utilis toutes sortes de techniques mathmatiques pour dvelopper des procdures danalyse de donnes requirent seulement des formules simples.

Apprendre la statistique classique se rsume donc connatre les recettes danalyse qui existent, et de savoir quelle recette utiliser dans quelle situation.

La statistique moderne#

Lutilisation de lapproche computationnelle simplifie grandement lapprentissage de la statistique. Un mouvement croissant dans lenseignement des statistiques, appel statistique moderne , met laccent sur lutilisation des ordinateurs pour les calculs statistiques. Les deux techniques du programme de statistique moderne que nous aborderons dans cet article sont:

  • Simulation: Approche computationnelle directe pour effectuer des calculs de probabilits en simulant des dizaines de milliers dchantillons de la population concerne. Il est possible de calculer les quantits pertinentes directement partir des chantillons, sans utiliser de formules mathmatiques.

  • Mthodes de rchantillonnage: Terme gnrique dsignant diverses techniques qui rutilisent les donnes dun chantillon observ pour simuler la variabilit de la population. Nous aborderons lestimation par bootstrap dans ce notebook.

Les mthodes de simulation et de rchantillonnage sont coteuses en calcul, mais elles sappliquent toute distribution et permettent de calculer toute quantit dintrt, et pas seulement celles pour lesquelles les statisticiens ont trouv des formules.

Estimation par bootstrap#

La mthode bootstrap est une technique puissante pour estimer la variabilit de tout estimateur.

Le bootstrap utilise une simple boucle for, qui gnre plusieurs reprises de nouveaux chantillons partir des donnes existantes. Cette approche de lestimation de lincertitude est trs intuitive et permet une comprhension plus approfondie que les mthodes dapproximation classiques bases sur des formules.

import numpy as np
from scipy.stats import norm
from ipywidgets import interact

def plot_pop_and_sample(mu, sigma):
    ax = sns.histplot(x=chantillon, stat='density', bins=range(50, 150, 5), label='data')
    rv_X = norm(mu, sigma)
    xs = np.linspace(50, 150, 1000)
    f_Xs = rv_X.pdf(xs)
    sns.lineplot(x=xs, y=f_Xs, ax=ax, c='C0', label='model')
    print(f'modle pour la population: X ~ N(mu={mu}, sigma={sigma})')

interact(plot_pop_and_sample, mu=(50, 150, 1), sigma=(3, 40, 1));

Moyenne de lchantillon#

def moyenne(xs):
    return sum(xs) / len(xs)

moyenne(chantillon)

105.89

La moyenne de lchantillon est une estimation pour la moyenne de la population \(\mu\).

cart-type de lchantillion#

Voici un autre estimateur: lcart-type \(s_{\mathbf{x}} = \tt{std}(\mathbf{x}) = \sqrt{ \frac{1}{n-1}\sum_{i=1}^n (x_i-\overline{\mathbf{x}})^2 }\).

import numpy as np

def std(xs):
    n = len(xs)
    xbar = moyenne(xs)
    s_2 = sum([(xi - xbar) ** 2 for xi in xs])
    return np.sqrt(s_2 / (n - 1))

std(chantillon)

14.658229417469641

Lcart type de lchantillon est une estimation de lcart type de la population \(\sigma\).

np.random.seed(46)
bmoyennes = []
for i in range(5000):
    bsample = np.random.choice(chantillon, n, replace=True)

with plt.rc_context({'figure.figsize': (8, 2.8)}):
